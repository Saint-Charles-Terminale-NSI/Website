Projet NSI Terminale

Bienvenue sur le projet NSI Terminale, ce projet à pour but de montrer vos compétences acquérit en informatique durant vos deux années de NSI.


Pour cela vous devrez réaliser un projet en groupe ou seul, comme vous le souhaitez. Ce projet peut se faire dans le langage de votre choix (si vous voulez utiliser un langage qui n'est pas vu en cours demander à votre prof).


Quant au sujet du projet c'est vous qui décidez de ce que vous voulez faire, la seule contrainte imposée est de faire un site web pour expliquer votre projet (page HTML et CSS).


Bonne chance pour vos projets, et si vous le souhaitez vous pouvez consulter les projets des années antérieurs ainsi que leurs codes sources sur la page Github.
