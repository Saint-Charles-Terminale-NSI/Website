Anagramme



C'est quoi une anagramme?
Une anagramme est un mot formé en changeant de place les lettres d'un autre mot. Exemple: Une anagramme de gare est rage.

Et il fais quoi le programme?
Comme son nom l'indique le programme cherche les anagrammes du mot que vous avez entré et vous les afficher.

Comment ça marche?
Le programme vous demande votre mot, vérifie le nombre d'anagramme de ce dernier (en utilisant les factorielles), si il est supérieur à 10000 le programme ne se lancera pas. Sinon le calcul d'anagramme se fait puis vous les affiches
